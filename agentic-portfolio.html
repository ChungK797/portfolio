AI-Powered Portfolio

Autonomous Agentic Development

Role: Architect & Director
Timeline: 2026
Tech Stack: HTML, CSS, JS, Agentic AI

Overview

I built this very portfolio website from scratch utilising advanced agentic AI. The goal was to leverage autonomous coding assistants to rapidly design, develop, and iterate on a modern, premium web experience without relying on bloated templates or frameworks.

The Process

Operating in an intensive pair-programming dynamic, I orchestrated the AI to construct the semantic HTML architecture and build out a robust, variable-driven CSS design system. We rapidly iterated on the aesthetics, seamlessly transitioning the entire site from a dark mode to a crisp light mode environment in minutes.

Technical Execution

  • Utilised natural language prompting to generate highly optimised, zero-dependency Vanilla JS and CSS.
  • Deployed autonomous browser subagents to navigate external live URLs, capture viewport screenshots, and save them directly to the local filesystem.
  • Leveraged native macOS command-line tools (`sips` and `awk`) via agentic bash execution to dynamically calculate dimensions and aggressively crop visual artefacts from screenshots.
  • Engineered custom routing fixes to intercept and strip URL hashes, overriding native browser scroll behaviours to guarantee top-of-page loads.

Outcomes

The result is a lightning-fast, highly responsive static website that perfectly encapsulates my professional identity. This project serves as a live demonstration of the immense power and efficiency of human-AI collaboration in accelerating the software development lifecycle.
