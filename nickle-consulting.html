Nickel Consulting

Website Redesign & WordPress Migration

Role: Team Lead
Timeline: 03/2025 – 06/2025
Tech Stack: WordPress, cPanel, CSS

Overview

I led a 5-person team in a comprehensive redesign of the Bickel Consulting website. The core objective was to migrate their existing infrastructure to WordPress, resolving underlying UI issues and implementing full mobile responsiveness for a seamless modern web experience.

Leadership & Process

To ensure a smooth delivery, I facilitated weekly sprints with the development team. I was responsible for translating business needs into technical solutions, which included presenting design prototypes directly to product owners to guarantee alignment before execution.

Technical Execution

  • Managed end-to-end WordPress configuration, integrating essential plugins and customising the theme.
  • Developed custom CSS to ensure strict brand alignment across all digital touchpoints.
  • Oversaw backend architecture and optimised performance utilising cPanel.
  • Executed rigorous cross-browser testing to guarantee quality and consistency.
  • Managed the complete end-to-end content migration process.

Outcomes

The successful launch resulted in significantly improved load speeds and enhanced navigation, leading to a marked increase in user engagement. Furthermore, the new WordPress infrastructure empowered the client with the ability to manage content updates completely independently, reducing ongoing maintenance friction.
